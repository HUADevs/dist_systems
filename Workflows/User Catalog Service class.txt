Class Diagram

Ακολουθούμε την λογική του MVC Pattern. Στο παραπάνω διάγραμμα διακρίνουμε τα δυο βασικά packages της εφαρμογής(Data,Controller). Το Data θα είναι αυτό που τα object του θα αποθηκευτούν στην βάση δεδομένων. Η κάθε κλάση ,δηλαδή, είναι ένας πίνακας και τα attributes είναι οι πλειάδες.
Όλα τα attributes διαθέτουν getters/setters επειδή είναι, σαν αντικείμενα, όλα private.

To package Controller έχει τις κλάσεις για κάθε πίνακα και όλες τις μεθόδους που αντιστοιχούν στις εκάστοτε ενέργειες του διαχειριστή.


Όλη η λογική των Views καλύπτεται άπο τα JSP & static files.
